# Message
调用 message(string) 后，页面上显示出一个漂浮的信息弹窗。效果参考 element 的 [message](https://element.eleme.cn/#/zh-CN/component/message) 方法

在 [index.js](./index.js) 和 [index.vue](./index.vue) 中进行代码的编写。组件测试在 [message.spec.js](./message.spec.js) 中。

## Feature
- 调用方法后弹窗出现
- 一段时间后自动消失

### High Feature
- 可插入除字符串之外的内容
- 鼠标悬浮时不会消失
- 点击后主动消失
- in/out 动画
- 可以显示多条message
- 修改或者添加组件测试
- 等等。。。
