# Swiper
在有限空间内，循环播放同一类型的图片、文字等内容
## Feature
- 自动轮播
- 点击按钮切换

### High Feature
- 外部控制当前展示
- 轮播内容由props或slot传入
- 大小自适应
- 性能优化
- 多种过渡方式
- 修改或者添加组件测试
- 等等。。。
